Quick Start

aria-label 속성에 발화를 원하는 텍스트를 넣으면 됩니다.
tabIndex로 포커스 가능한 요소에 aria-label을 작성하면, 해당 요소가 포커스될 때 텍스트가 자동으로 음성 변환됩니다.
가장 기본적인 사용법TypeScript
// tabIndex로 포커스 가능하게 만들고, aria-label에 읽어줄 텍스트를 넣습니다.
<button
  tabIndex={0}
  aria-label="층별안내: 확인 버튼을 누르시면 층별 안내로 이동합니다."
>
  층별안내
</button>

이것이 전부입니다. tabIndex로 포커스 가능한 요소에 포커스가 들어오면, aria-label에 작성된 텍스트가 음성으로 재생됩니다.

tabIndex로 포커스 가능 → 요소에 포커스 진입 → aria-label 텍스트 추출 → TTS 음성 재생
TTS 엔진포커스된 요소의 aria-label 처리 방식
Windows 내레이터tabIndex로 포커스된 요소의 aria-label을 Windows 스크린 리더가 자동으로 읽음
로컬 TTS (ONNX)tabIndex로 포커스된 요소의 aria-label 텍스트를 추출하여 로컬 TTS 엔진에 전달 후 음성 생성

개발자가 하는 일은 tabIndex와 aria-label을 작성하는 것뿐이고, 나머지는 TTS 엔진이 자동으로 처리합니다.

선택 상태가 있는 버튼

동적 상태 반영TypeScript
const MenuButton = ({ name, selected }: { name: string; selected: boolean }) => (
  <button
    aria-label={`${name}${selected ? ', 선택됨' : ''}: 확인 시 이동합니다.`}
  >
    {name}
  </button>
);

// selected=true  → "층별안내, 선택됨: 확인 시 이동합니다."
// selected=false → "층별안내: 확인 시 이동합니다."

시설 목록

리스트 렌더링TypeScript
interface Facility {
  id: string;
  floor: number;
  name: string;
}

const FacilityList = ({ items }: { items: Facility[] }) => (
  <>
    {items.map((item) => (
      <div
        key={item.id}
        aria-label={`${item.floor}층 ${item.name}, 확인 시 길 안내를 시작합니다.`}
        tabIndex={0}
      >
        {item.name}
      </div>
    ))}
  </>
);

// → "3층 대회의실, 확인 시 길 안내를 시작합니다."

초기 안내 가이드

가이드 메시지TypeScript
const guideLabel = [
  '안녕하세요. 키오스크 사용을 도와드리겠습니다.',
  '상하좌우 방향키로 항목을 이동할 수 있습니다.',
  '확인 버튼을 누르면 선택됩니다.',
].join(' ');

<div aria-label={guideLabel} tabIndex={0}>
  키오스크 안내
</div>

// → "안녕하세요. 키오스크 사용을 도와드리겠습니다. 상하좌우 방향키로 항목을 이동할 수 있습니다. 확인 버튼을 누르면 선택됩니다."

label 생성 함수 분리

label 텍스트가 복잡하거나 여러 곳에서 재사용된다면 함수로 분리할 수 있습니다.

utils/labels.tsTypeScript
export const floorLabel = (floor: { name: string; facilities: string }): string =>
  `${floor.name} 안내입니다. 이 층에는 ${floor.facilities}이 있습니다.`;

export const menuLabel = (name: string, selected: boolean): string =>
  `${name}${selected ? ', 선택됨' : ''}: 확인 시 이동합니다.`;
사용TypeScript
import { floorLabel, menuLabel } from '../utils/labels';

<div aria-label={floorLabel(currentFloor)}>...</div>
<button aria-label={menuLabel('층별안내', true)}>층별안내</button>
